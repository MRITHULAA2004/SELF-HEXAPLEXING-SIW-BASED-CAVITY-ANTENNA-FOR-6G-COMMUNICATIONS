# SELF-HEXAPLEXING-SIW-BASED-CAVITY-ANTENNA-FOR-6G-COMMUNICATIONS
Design and analysis of a compact SIW-based self-hexaplexing antenna for sub-6 GHz applications

📡 SIW-Based Self-Hexaplexing Antenna

🔹 Overview

This project presents the design and simulation of a compact self-hexaplexing antenna based on a hexagonal Substrate Integrated Waveguide (SIW) cavity. The antenna supports multiple frequency bands using slot-based resonators.

🔹 Key Features

Multi-band operation using a single compact structure
Independent frequency tuning using slot geometry
High port isolation between resonating elements
Suitable for sub-6 GHz communication systems

🔹 Tools Used

CST Studio Suite
Origin Pro

🔹 Results

Multiple resonant frequencies achieved
Good impedance matching
Stable radiation characteristics

🔹 Applications

Wireless communication systems
Multi-band RF front-end modules
Compact transceivers
